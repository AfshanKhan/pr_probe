# Contributing to PR-Probe

Thank you for your interest in improving PR-Probe! We welcome all contributions.

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally.
3.  **Create a branch** for your changes: `git checkout -b feature/my-new-feature`.
4.  **Install dependencies** using `uv`:
    ```bash
    uv sync
    ```
5.  **Make your changes** and ensure the code follows the existing style.
6.  **Run the tool** locally to verify your changes:
    ```bash
    uv run pr-probe --help
    ```
7.  **Commit your changes** with a clear message.
8.  **Push to your fork** and **submit a Pull Request**.

## Environment Setup

You will need a GitHub Personal Access Token (PAT) with `repo` scopes to test the tool against private repositories.
